Reborg Bugs

Problem Description

A group of engineers designed a little robot called reborg that can turn left, right and move in one direction. Assume there is a plane with infinite grids, horizontally and vertically, and the engineers place reborg at grid (0,0) facing north.

Reborg can recieve the following instructions:

  • "G": go straight 1 unit;
  • "L": turn 90 degrees to the left;
  • "R": turn 90 degrees to the right.

Reborg performs the instructions given in order, and repeats them forever.

Reborg is having bugs
if and only if there exists a loop on the plane such that Reborg never leaves the loop.

Given the instructions, return if reborg has bugs.

Example

Input 1: “GGRRGG”
Output 1: true
Explanation 1: Reborg moves from (0,0) to (0,2), turns 180 degrees, and then returns to (0,0). When repeating these instructions, the robot remains in the circle of radius 2 centered at the origin.
Input 2: “GGGG”
Output 2: false
Explanation 2: Reborg keeps moving north forever, no loop will occur.

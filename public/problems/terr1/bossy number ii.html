Bossy Number II

Problem Description

You might know some numbers are quite bossy, but some numbers are even bossier. A number a is bossier if and only if:

1- The number is a positive integer.
2- It has only 2 factors.

Given a number a, return if a is a bossier number.

Example

Input 1: a = 15
Output 1: false
Explanation 1: 15 is not a bossier number because its factors are 1,3,5,15, the number of factors is 4, therefore return false.
Input 2: a = 37
Output 2: true

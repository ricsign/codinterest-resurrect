Exclamation Marks

Problem Description

“What is the answer of 1 X 2 X 3 X 4 X 5 X 6 X 7?”
“7!”
“No, how is that 7?”
“It’s not 7, it’s 7!”
“How?”
…

This is not an ELA lesson where you need to analyze a character’s tone. In mathematics, the exclamation mark represents factorial. The informal mathematical definition looks like this:

n! = n*(n-1)*(n-2)*…*3*2*1

In other words, n! equals the product of all positive integers less than or equal to n.

Given a value n, you will be asked to find the result of 1! + 2! + … + (n-1)! + n!

The result may be huge, output the result by taking the modulo of 10007. (If you’re not sure about modulo operations, Click Here to find out more.

Example

Input 1: n = 4
Output 1: 33
Explanation 1: Our goal is to find 1! + 2! + 3! + 4!, or 1 + 1*2 + 1*2*3 + 1*2*3 + 1*2*3*4, or 1 + 2 + 6 + 24 = 33, 33 % 10007 = 33, return 33.
Input 2: n = 8
Output 2: 6205

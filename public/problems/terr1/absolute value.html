Absolute Value

Problem Description

The absolute value of a number x is defined as following:

1- If the number is nonnegative, return x itself.
2- Otherwise return the value of 0-x.

Example

Input 1: x = 20
Output 1: 20
Input 2: x = -9.85
Output 2: 9.85

Testcases

Number of Testcases: 7
Specification: Every testcase consists one line, each line has a single number x, output the absolute value of x.
Input:
1
-1
19
-9.989
0
1.001
-123456.789

Expected Algorithms

  • Conditionals

Expected Time Complexity & Space Complexity

Time Complexity: O(1)
Space Complexity: O(1)

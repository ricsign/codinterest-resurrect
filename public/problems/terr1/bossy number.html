Bossy Number

Problem Description

Like us, some numbers are quite bossy. A number a is bossy if and only if:
1- The number is a positive integer.
2- It has odd number of of factors.

Given a number a, return if a is a bossy number.

Example

Input 1: a = 12
Output 1: false
Explanation 1: 12 is not a bossy number because its factors are 1,2,3,4,6,12, the number of factors is 6, 6 is an even number, therefore return false.
Input 2: a = 36
Output 2: true

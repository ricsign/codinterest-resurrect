Chameleon Numbers

Problem Description

An n-digits Chameleon Number satisfies the following:

  1. All digits consist only one’s and two’s
  2. None of two adjacent digits that made up Chameleon Number are made of one’s.

Given n, find the number of n-digits Chameleon Numbers taking the modulo of 10000007.

(For instance, 1221 is a 4-digits chameleon number, while 112 is not a 3-digits chameleon number since the first digit and the second digit from left to right are both one’s.)

Example

Input 1: n = 2
Output 1: 3
Explanation 1: All two-digits Chameleon Numbers are 12,21,22.
Input 2: n = 10
Output 2: 144
